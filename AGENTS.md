# Welcome, Agents

This cross-platform desktop application is about testing HTTP APIs. Supported protocols include REST, WebSocket, GraphQL and gRPC.

## General

1. Always ask before making changes if you have anything unclear.
2. Compose Desktop 1.6 and Kotlin are used. Do not upgrade Compose beyond 1.6.
3. Read [Code Practice](CODE_PRACTICE.md) for coding practice. You will need to follow it strictly.
